# SFL-PAPER-REPLICATION-PACKAGE
R and stata code for SFL working paper by Glassow & Kirsten (working paper version).

This repository contains .R and .do files that provide the code for data preparation, analysis, and plots for the paper: Does in-service training in special education increase the effectiveness of general classroom teachers? Evidence from the large-scale “Special Education Pedagogy for Learning” program in Sweden by Leah Glassow and Nils Kirsten, which is currently a working paper. 

Contents
R files for data preparation and plots, stata do files for csdid analysis in stata, and the output from stata. The micro-level data are not available publically as they are register data. You can apply for access to the data from Statistics Sweden. 

Requirements
Before running the analysis, ensure you have the following:

R 
tidyverse
janitor
did
tableone
fixest
haven
foreign 

Stata
csdid 

Load Packages: Loads necessary R packages.
Load and Clean Data: R
Run analysis: Stata 
Plot effect sizes: R
